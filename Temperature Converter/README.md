# 🌡️ Temperature-Converter
1-Month Software Engineering Internship at SkillCraft Technology | First Task: Temperature Converter Project 🚀

A modern and responsive web application that converts temperatures between **Celsius**, **Fahrenheit**, and **Kelvin** scales instantly.

This project is built using **HTML**, **CSS**, and **JavaScript** and is designed as a beginner-to-intermediate level frontend development project suitable for:

* Software Development Internship
* GitHub Portfolio
* Mini Project Submission
* Frontend Practice

# 📚 Temperature Conversion Formulas

## Celsius to Fahrenheit

```text
(°C × 9/5) + 32
```

## Fahrenheit to Celsius

```text
(°F − 32) × 5/9
```

## Celsius to Kelvin

```text
°C + 273.15
```
